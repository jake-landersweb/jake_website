About Me

My name is Jake Landers, and I am a fullstack developer based in Portland, OR. I publish professional tutorial format articles for people to learn amazing technologies that enable their project or business to thrive.

Here are a few things I use:

  • SwiftUI
  • I learned how to code composing SwiftUI iPhone apps, and continue to use the framework to create apps for iPhone, iPad, and mac
  • Flutter
  • I use Flutter to create most iOS apps, and all Android apps. Fast and efficient tool for mobile development
  • NoSQL
  • This was the first database system I learned to use, and mainly use AWS DynamoDB. But, I do use a self hosted MongoDB as well.
  • Django
  • Python is my favorite language, and as such Django is the framework I use for more statically driven websites, such as this one!
  • React
  • I use React when a website needs a strong concept of state and lots of user interaction.
  • PyTorch
  • When I hjave a need for machine learning, I use the PyTorch library over TensorFlow. (Although now both are pretty similar).
